# Self Service & Automation

#### Knowledge Management
A system for creating, storing, and sharing information articles.

Purpose:
- Enable self-service
- Reduce support workload
- Provide answers to users

#### Service Catalog
A self-service area where users request services, products, or assistance.

Examples:
- Hardware request
- Access request
- Leave request

#### Catalog Item
An item available in the Service Catalog that users can request.

#### Record Producer
A type of catalog item that creates a record directly in a table.

Example:
A Record Producer creates an Incident record.

#### Workflow Studio
A tool used to design and automate business workflows.

#### Flow Designer
A low-code automation tool used to create flows with triggers and actions.

#### Virtual Agent
A chatbot capability that helps users solve problems and complete requests through conversations.

#### Automation
The process of automatically performing tasks using workflows, flows, scripts, and integrations.
