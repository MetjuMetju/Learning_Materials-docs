OPENAI API USAGE
-----------------
https://platform.openai.com/api-keys

Purpose:
- Create API keys for OpenAI access
- Use GPT models via terminal or scripts
- Build automation or AI tools

Example CURL request:

curl https://api.openai.com/v1/chat/completions \
-H "Content-Type: application/json" \
-H "Authorization: Bearer $OPENAI_API_KEY" \
-d '{
  "model": "gpt-4o-mini",
  "messages": [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Write a haiku about Ubuntu CLI"}
  ]
}'

Result:
- JSON response with AI generated text


DDGR CLI SEARCH TOOL
--------------------
Command:
ddgr "linux kernel 6.11 release notes" --num 5 --safe off

Description:
- DuckDuckGo terminal search tool
- Provides web results in CLI
- Lightweight and fast

Parameters:
ddgr        = run search tool
query       = search text
--num 5     = limit results
--safe off  = disable safe search

Use cases:
- search web without browser
- server environments
- terminal workflows
