    Platform Overview and Navigation

    What is ServiceNow?

    ServiceNow is a cloud-based platform that provides digital workflows for IT, employees, customers, and business processes.

    ServiceNow Instance

    A unique ServiceNow environment with its own database, applications, users, configurations, and customizations.

    Examples: - Development instance - Test instance - Production instance

    Application

    A collection of modules, tables, processes, and functionality designed for a specific business purpose.

    Module

    A navigation item that opens a specific page, list, form, or function inside an application.

    Application Navigator

    The navigation panel used to find and access applications, modules, and platform features.

    Lists

    A view that displays multiple records from a table.

    Forms

    A page used to view, create, or modify a single record.

    Record

    A single row of data stored in a table.

    Next Experience

    The modern ServiceNow user interface providing unified navigation, personalization, and role-based experiences.

    Workspace

    A specialized interface designed for users working with specific processes, tasks, or roles.

    Global Search

    A search function used to find records, applications, knowledge articles, and other platform information.
